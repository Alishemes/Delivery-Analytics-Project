# Delivery Performance Dashboard

## Overview
The **Delivery Performance Dashboard** is an interactive data analysis project that visualizes real delivery trip data to provide daily performance insights for branches and drivers. It helps track key metrics and supports data-driven decision-making.  

## Key Features
- Total trips and total orders analysis  
- Average orders per trip  
- Percentage of single-order trips  
- Daily performance trends across the month  
- Branch-level analysis of trips and average delivery time  
- Driver performance evaluation based on trips and orders  
- Delivery method analysis (Car – Bike – Other)  

## Dashboard Highlights
- Fully interactive filters (day, year, branch)  
- Clear visual trends for daily performance  
- Branch and driver comparisons across multiple KPIs  
- Comprehensive overview of delivery time and efficiency  
- Modern, user-friendly design
